# HelpManager
Sitema de Help Desk e Gerenciador de Inventário — SPA em React + TypeScript com Tailwind e Radix UI

 java -version  
 mvn -v    
 ./mvnw spring-boot:run 
 npm install     
 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process     
 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
 npm -v   

Rodar Front : 
 npm run dev

 Rodar Back : 
 ./mvnw spring-boot:run
